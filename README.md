# Crypto Trader

A full-stack crypto trading agent for wallet management, DEX/CEX trading, market intelligence, and social signals across 20+ chains.

## Capabilities

**Wallet & Trading**
- Multi-chain wallet (Ethereum, Base, BSC, Arbitrum, Solana, and 15+ more)
- DEX token swaps with automatic security scan before every trade
- CEX spot & futures trading (Binance, Bybit, OKX, Hyperliquid)
- DeFi deposit/withdraw (Aave, Lido, Compound, and more)

**Market Intelligence**
- Real-time token prices, K-lines, gas fees
- Token research: metadata, holder distribution, trending rankings
- Smart money, whale, and KOL on-chain signals
- Meme token scanning with rug pull and bundle detection

**Social & News**
- Twitter/X KOL monitoring and real-time event tracking
- Crypto news with AI impact ratings and trading signals

**Portfolio & Security**
- Multi-chain portfolio tracking
- DeFi position overview
- Token risk detection before every swap

## Disclaimer

This agent is a tool, not a financial advisor. All trading decisions are yours. Crypto markets are volatile — only trade what you can afford to lose.
